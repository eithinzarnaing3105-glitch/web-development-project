# CineGrid - Movie Website
## Description
CineGrid is a web-based movie discovery platform that allows users to browse, search, and explore movie information through an intuitive and interactive grid layout. The website provides detailed movie information, trailers, ratings, and user-friendly navigation.

## Features
- Browse movies in a responsive grid layout
- Search movies by title, genre, or year
- View detailed movie information (plot, cast, ratings, etc.)
- Watch trailers
- Responsive design for mobile and desktop

## Technologies Used
- HTML
- CSS
- JavaScript
- VS Code

## How to Run
1. Download or clone the repository
2. Open the files in web browser
